-- INTRODUCTION --

Woocommerce GlobalPay allows payments to be made to a Woocommerce shop via
GlobalPay.

GlobalPay is a web-based payment gateway that enables merchants with functional
websites to accept card payments from customers worldwide. 

Requires at least: Woocommerce 2.0.14
